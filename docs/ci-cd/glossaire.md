# Glossaire CI/CD, Docker et Kubernetes

## Docker
Outil permettant de créer et exécuter des conteneurs.

## Image Docker
Modèle utilisé pour créer des conteneurs.

## Registre
Serveur qui stocke des images Docker.

## GHCR
GitHub Container Registry.

## Kubernetes
Plateforme d’orchestration de conteneurs.

## k0s
Distribution légère de Kubernetes.

## Cluster
Ensemble de machines exécutant Kubernetes.

## Pod
Plus petite unité déployable dans Kubernetes.

## Deployment
Objet Kubernetes décrivant comment déployer une application.

## Service
Objet Kubernetes exposant une application.

## Ingress
Objet Kubernetes gérant l’accès HTTP/HTTPS.

## kubeconfig
Fichier permettant à kubectl de se connecter à un cluster.

## kubectl
Outil en ligne de commande pour Kubernetes.

## Runner self-hosted
Machine exécutant les workflows GitHub Actions.

## Workflow
Fichier YAML décrivant les étapes CI/CD.

## Secret
Valeur sensible stockée dans GitHub.
